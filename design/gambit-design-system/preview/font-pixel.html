結論建議
❌
點陣體當標題 — 完全改變品牌感，跟「精緻木質棋藝殿堂」定位衝突。除非你想把整個品牌方向改成「復古像素棋盤遊戲」，否則不建議。
✅
點陣體用在棋譜 + eval 數字 — 非常有個性！有老式棋盤電腦的 terminal feel，是對棋類文化的一種致敬。可以取代原本規劃的 Sarasa FixedSlab TC，效果甚至更獨特。
💡 推薦最終組合：標題 BIZ UDPMincho · 品牌 Cinzel · UI Sarasa UI TC · 棋譜/數字 俐方體 Cubic 11
這個組合把「精緻不老氣」和「棋類個性」都照顧到了。
