Board Frame Exploration — 4 Directions

差異化但不分心 — 每個方向都能靜靜存在，不搶走棋局注意力

01 深木雕框
加深木紋框，三維感來自角度漸變 + 頂亮底暗邊緣。像實體高端棋盤的外框，觸感感強。
✓ 最有實體感 ✓ 跟品牌暖色系一致 △ 跟 Chess.com 木框方向接近
02 細金絲角飾
棋盤本身幾乎無框，只在四個角落放極細的金色裝飾線。整體非常克制，角飾像古典西洋棋盒上的金屬扣件。
✓ 最獨特，市面上找不到類似 ✓ 完全不搶注意力 ✓ 呼應品牌金色
03 青玉石框
框架直接使用品牌青玉色，把棋盤和整體 UI 語言連成一氣。底部細金點是 file 座標的抽象化。
✓ 品牌一致性最強 ✓ 跟其他 app 明顯不同 △ 棋盤區和 UI 界線略模糊
04 沉漆金線框
幾乎全黑的漆器效果，框內有一圈極細金線（0.25px 的品牌金色 inner rim）。頂部有琥珀色漆光，像真正的日本漆器棋盒。
✓ 最 premium 感 ✓ 日本傳統工藝美學 △ 比木框更深，整體偏暗
設計師建議
我最推 02 細金絲角飾
市面上完全沒有人這樣做。棋局進行時角飾退到背景感知不到，但靜下來看的時候會覺得「這個棋盤很精緻」。 這種「低存在感，高質感」的設計語言跟你整個品牌方向完全一致。
備選 02+01 混搭
金絲角飾 + 極薄木框（3-4px），兩者同時存在不衝突。角飾提供識別度，薄木框給棋盤一點溫度和邊界感。 這樣差異化最大，同時維持實體棋盤的溫暖感。
